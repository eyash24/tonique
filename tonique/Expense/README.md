# Expense

## About
This module consist of functions and objects that can be used in financial expense setting. It can be utilsied for calculating the splt expense of group members. It is a simplified version of Splitwise.

A sample workflow can be found in the within the module. \
Sample file: [Expense sample](Sample/Expense_sample.py)

## Limitation 
- Currenlty the module is able to only add members into a group object and calculate transactions split within that group.
- Support for overall split (for multiple group) and persistence storage for transactions are under development.
